Simple Process Mining

By utilizing the power of processing and simplifying usability through clear visualizations, we enable Journey Owners to effortlessly identify anomalies, streamline their operations and ultimately be provided with adapted data based on their modifications.


Overview of the tool

Initial Situation

Swisscom's process of analyzing user journey data is complex and time-intensive, currently requiring highly specialized personnel. They need a streamlined process mining solution to simplify and visualize the customer journey, enabling the efficient identification of key processes for optimization.

Project Goal

The focus is on minimizing the complexity of journey data analysis to enhance distributed decision-making within the Journey Analysis team. By doing so, the analysts can more independently optimize journeys, thus reducing the costs associated with customer interactions. Along the way, it is our internal goal to learn more about Scrum and individually grow as developers and project members.

Home Screen

Here you can see a preview of our home screen where users will be able to add and edit leads, as well as search and filter for specific leads.

Settings Page

On the settings page users can be managed (add, edit, delete) and organized into teams and general agencies. Additionally, certain attributes of leads can be configured.

Import Page

Users can import data directly from an excel file and then match the columns to the given attributes from the leads table on the home screen.
