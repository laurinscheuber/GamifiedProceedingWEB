Power System Monitoring

A smart web app for power system monitoring: visualize data, manage units, connect devices, and integrate seamlessly with a standardized API for efficient and customizable operation.

Renato's and Vipluv's research group started developing a flexible IoT data analytics platform to modernize power electronic systems. These systems, essential for electric conversion in applications such as electric vehicles, wind turbines, and motor drives, aim to use predictive analytics to reduce inefficiencies and downtime. The platform features dataloggers for data collection and a cloud-based setup for advanced analysis.
Despite these advancements, integrating and monitoring the systems remains challenging. The data visualization and system diagnostics rely on tools that are not user-friendly. Additionally, implementing predictive fault detection requires better machine learning integration and improved visualization tools, highlighting the need for a more efficient and intuitive solution.

Project Goal

The goal of the PowerSystemMonitoring project is to on build a comprehensive monitoring tool designed to gather data from a variety of devices and provide visual insights into their operational status. At the heart of the platform are two key elements: a scalable deployment infrastructure and a standardized API. This API plays a pivotal role in enabling seamless data collection from multiple device types, channeling this data into a central repository (data lake) for further analysis and visualization of device health.

Developed Solution and its benefit

The developed solution is a web application designed to provide a comprehensive overview of units and the status of connected devices. Users can monitor the state of their units and devices, add new units, connect devices, and create personalized visualizations. For developers, the standardized API simplifies the integration of new data, ensuring seamless communication and flexibility.

  • Visualization of state and time-series data: Users can easily visualize both state and time-series data, with seamless data transmission to the frontend for analysis.
  • Customizable visualizations: The platform allows users to tailor visualizations, enabling them to prioritize and identify more urgent leads effectively.
  • Creating templates for visualizations: Users can design templates specifying the types of graphs and data they want to visualize, streamlining the customization process.
  • Adding new units and devices: The system makes it simple to add new units and devices, as well as connect devices to existing units effortlessly.
  • Standardized API: The standardised API provides documentation that makes it easy to send a wide variety of data to the frontend, which makes the application very flexible.
