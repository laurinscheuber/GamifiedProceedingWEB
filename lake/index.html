Initial Situation/Problem statement

The "Shake the Lake" summer event hosted by ti&m, is the once a year event that draws numerous attendees eager to enjoy boat rides and lakeside activities. Unfortunately, the current organizational setup heavily relies on manual processes, creating significant inefficiencies.

Customer information, including credentials, is collected at an information hut and written down by hand, resulting in long wait times and potential errors in data handling. Changes to boat schedules are also challenging to manage. Without a centralized system, rescheduling or notifying impacted customers about schedule changes is inefficient and oftentimes almost impossible to do, leading to confusion and dissatisfaction. Overall, the reliance on pen-and-paper processes creates an administrative overhead that detracts from the overall event experience for both attendees and organizers.

Project Goal

The aim is to achieve an Shake the Lake event experience that is uncomplicated and simple for attendees and the organisation committee. Achieving this goal will be done with a mobile app for attendees and a web application for the organization committee.

Attendees will benefit from a user-friendly mobile app that simplifies the booking process, enabling them to easily select their preferred boat rides and activities. The app will also send timely reminders and updates, ensuring they are informed about their schedules and any changes.

The organisation committee will have access to a web app designed to optimize administrative planning. This platform will allow organizers to send custom notifications to attendees, assist with manual booking entries for customers without access to the mobile app, and monitor bookings and reschedule timeslots efficiently. By centralizing and automating key processes, the system will significantly reduce manual overhead while enhancing the overall experience for both attendees and organizers.

Admin web app

Event overview

This is the event overview page for the organisation committee. All basic event information is shown on this page and it can be edited here.

Timetable

On the timetable view the organisation committee can see what boats exist and which timeslots they have. The timeslots can be opened in a detail-view via clicking on them. This is the most important page for the organisation committee.

Timeslot - Detailview

On the timesot detailview additional information of that slot can be viewed. Additionally it can be edited, in the case of schedule or similar changes.

Booking - Detailview

Manual bookings can be done aswell. If a attendee has no mobile access, the organisation committee can book a timeslot manually for them.

Boat - Detailview

The organisation committee can create and edit boats in this view so they're available for the event.


Mobile application

Landing page

Overview of activities

Booking page

Developed solution and its benefit

The Shake the Lake application is an digital platform consisting of a mobile app for attendees and a web application for the organisation committee. This system modernizes and automates the booking and scheduling process, significantly reducing manual work while enhancing the overall experience for both guests and event staff.

  • Simple timeslot booking for attendees: Event attendees can easily book boat rides and other lakeside activities through the mobile application by providing all necessary details on their own mobile phone. This eliminates long queues and reduces the need for in-person registration.
  • Centralized scheduling updates: The mobile app sends reminders and instant notifications about schedule changes, keeping attendees informed and minimizing confusion.
  • Administrative efficiency for organizers: The web app enables the organisation committee to manage bookings, make manual entries for guests without mobile access and reschedule timeslots seamlessly.
  • Custom communication: The organisation committee can send customized schedule updates to attendees, ensuring timely and effective communication.
  • Overview of bookings: The web app provides access to booking statuses and schedules. As well as providing insight into who booked into which events.

The benefits of the developed solution provide a robust and stable event managment by making the registration process easier, faster and less prone to human error. Communication between organisation committee and the attendees has been made easier by having one centralized location to send notifications out to the attendees.
